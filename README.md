# HorizonTechX Superstore Dashboard

## Project Overview

This project is an interactive Superstore Sales Dashboard developed using Streamlit, Python, and Plotly as part of the Horizon TechX AI & Data Science Internship Program.

The dashboard converts raw sales data into meaningful business insights using interactive charts, KPI metrics, filters, and analytical visualizations.

The project helps analyze:
- Sales performance
- Profit trends
- Customer segments
- Regional analysis
- Product performance

# Installation

## Clone Repository

```bash
git clone https://github.com/varshithreddyy6/HorizonTechX_Superstore_Dashboard.git
```

## Open Project Folder

```bash
cd HorizonTechX_Superstore_Dashboard
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python -m streamlit run app.py
```
